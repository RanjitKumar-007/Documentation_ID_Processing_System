The input folder holds sample identity documents — Aadhaar cards, PAN cards, Voter IDs, Driving Licences, and Passports — in both PDF and image formats (.pdf, .png, .jpeg, .jpg), used to test the ID Processing pipeline's extraction, classification, and confidence scoring. Multi-page PDFs (e.g. Aadhaar front and back) are supported, with each page sent to the pipeline as a separate image; low-quality scans below the blur threshold are rejected before extraction. These files contain legally sensitive PII and should be used only for pipeline testing, not stored or shared beyond that scope.
